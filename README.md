# Data Structures Library System

A library management system developed in C++ as part of the Data Structures course at Sakarya University.

## Overview

This project demonstrates the implementation of multiple data structures within a library management environment.

## Features

- Book management
- User management
- Comment system
- Book rating system
- Request handling

## Data Structures Used

- Stack
- Queue
- Circular Linked List
- Hash Table
- Tree Structure

## Technologies

- C++
- Object-Oriented Programming
- Data Structures

## Project Type

University Project
